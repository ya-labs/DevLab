# 01 - Boas-vindas

Seja bem-vindo(a) ao DevLab.

Este espaço foi criado para estudar desenvolvimento de forma prática, organizada e próxima do que acontece em projetos reais.

Aqui, o objetivo não é apenas copiar código ou decorar comandos. A ideia é aprender a pensar como dev:

- entender problemas;
- quebrar problemas grandes em partes menores;
- escrever código com clareza;
- organizar arquivos;
- usar ferramentas reais;
- pedir ajuda do jeito certo;
- revisar erros e melhorar com o tempo.

## Como vamos estudar

O estudo será dividido em semanas.

Cada semana pode ter:

- aula ao vivo;
- material de leitura;
- exemplos práticos;
- exercícios;
- entrega;
- feedback.

Você não precisa entender tudo de primeira. Programação é construída com repetição, prática e revisão.

## O que eu espero de você

Durante os estudos, tente manter estes hábitos:

| Hábito | Por que isso importa |
| --- | --- |
| Anotar dúvidas | Ajuda a perceber exatamente onde você travou. |
| Tentar antes de pedir resposta | Desenvolve autonomia. |
| Explicar o que tentou | Facilita receber ajuda de verdade. |
| Praticar durante a semana | Programação melhora com constância. |
| Revisar erros | Erro é parte normal do processo. |

## O que não precisa acontecer agora

Você não precisa:

- saber tudo de JavaScript;
- dominar GitHub;
- decorar comandos;
- criar um projeto perfeito;
- ter vergonha de perguntar.

O começo serve para criar base.

Com base boa, o avanço depois fica muito mais natural.
