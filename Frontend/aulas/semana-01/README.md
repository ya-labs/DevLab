# Semana 01 - Início da Jornada

Bem-vindo(a) à primeira semana de estudos do DevLab.

Nesta semana, vamos entender como as aulas serão organizadas, por que vamos começar com JavaScript, como vamos usar o GitHub e como preparar o VS Code para começar a programar.

## Objetivo da semana

Ao final desta semana, você deve entender:

- como a mentoria vai funcionar;
- como usar o Discord durante os estudos;
- por que JavaScript será nossa primeira linguagem;
- para que serve o GitHub no nosso processo;
- como preparar o VS Code;
- qual será sua primeira atividade.

## O que você precisa fazer

1. Participar da aula ao vivo.
2. Ler os materiais da semana.
3. Configurar o ambiente.
4. Fazer a atividade proposta.
5. Enviar sua entrega no canal combinado do Discord.

## Importante

Esta primeira semana não é uma prova.

O objetivo é entender o ponto de partida de cada pessoa, organizar o ambiente e começar a criar uma base sólida para programar com mais segurança.
