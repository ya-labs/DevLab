# 02 - Como as aulas vão funcionar

As aulas serão organizadas para misturar explicação, prática e acompanhamento.

A ideia é evitar aquele estudo em que a pessoa só assiste conteúdo, mas não aplica nada. Aqui, você vai estudar e praticar.

## Fluxo da semana

O fluxo principal será este:

1. Aula ao vivo para apresentar o conteúdo.
2. Leitura dos materiais no repositório.
3. Exercícios ou atividade prática.
4. Entrega no Discord.
5. Revisão das dificuldades.
6. Feedback para ajustar o próximo passo.

## Como pedir ajuda

Quando você tiver uma dúvida, evite mandar apenas:

> Não entendi.

Prefira mandar algo assim:

```txt
Estou tentando entender algoritmos.
Eu entendi que algoritmo é uma sequência de passos, mas travei na hora de transformar isso em código.
Li o material X e tentei fazer o exercício Y.
Minha dúvida é: como eu sei qual passo vem primeiro?
```

Isso ajuda muito mais, porque mostra onde está a dificuldade.

## Como serão as entregas

As entregas podem começar simples.

Dependendo da atividade, você poderá entregar:

- mensagem no Discord;
- arquivo Markdown;
- arquivo de texto;
- PDF;
- link do GitHub;
- print, quando fizer sentido.

O mais importante no começo é mostrar seu raciocínio.

## Ritmo das aulas

O ritmo será ajustado conforme a turma.

Se muita gente travar no mesmo ponto, vamos revisar antes de avançar. O objetivo não é correr conteúdo. O objetivo é aprender com consistência.
