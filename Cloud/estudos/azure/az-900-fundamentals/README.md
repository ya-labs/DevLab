# AZ-900 - Microsoft Azure Fundamentals

## Objetivo

Estudar os fundamentos de cloud computing e dos principais serviços do Microsoft Azure.

## Blocos da prova

1. Conceitos de nuvem
2. Arquitetura e serviços Azure
3. Gerenciamento e governança

## Plano de estudo

- Dia 1: Conceitos de nuvem
- Dia 2: Arquitetura do Azure
- Dia 3: Computação, rede e armazenamento
- Dia 4: Identidade, acesso e segurança
- Dia 5: Gerenciamento, governança e custos
- Dia 6: Monitoramento, revisão e simulados
